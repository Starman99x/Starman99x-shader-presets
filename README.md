These shader presets are to be used with RetroArch, I do not take credit for most graphics used, refer to each included credits.txt for details.
The original shaders that these presets reference and depend on are:
- koko-aio by kokoko3k.
- Mega Bezel by HyperspaceMadness. 

**How to use:**
- Go to [releases]() and pick a pack of your choice.
- Unpack into: RetroArch/shaders.
- The aspect ratio must be set to "FULL" in "Settings/Video/Scaling".
- With Mega Bezel only, you may need to go into "Settings/Core/Allow Rotation" and set that to "OFF".
- Load a shader preset from "Quick Menu/Shaders" and go.
