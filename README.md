These shader presets are to be used with RetroArch, I do not take credit for all graphics used, refer to each included credits.txt for details.

**How to use:**
- Go to [releases](https://github.com/Starman99x/koko-aio-presets-and-overlays/releases) and pick a pack of your choice.
- Unpack into: RetroArch/shaders.
- The aspect ratio must be set to FULL in settings/video/scaling.
- Load a shader preset from Quick Menu/Shaders and go.
